    <h1 style="color:blue;"><p align="center">¿Que es el Software?</p></h1>
    <p>El software es todo lo que no se puede tocar en una computadora, es decir, los programas y aplicaciones que usamos para que el equipo funcione. Por ejemplo, el sistema operativo, las aplicaciones como Word o los juegos forman parte del software. Básicamente, es lo que le da instrucciones al hardware (las partes físicas) para que haga lo que nosotros queremos. Sin el software, una computadora no podría hacer nada útil.</p>
    <img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRBPDDarv26eUtM-6WgtMbuYz-5SdxzNWn99E5Oq91oOg&s=10">
    <h2 style="color:black;"><p>Caracteristicas del Software</p></h2>
<p>■Es intangible, es decir, no se puede tocar porque no es algo físico.<br>
    ■Funciona mediante instrucciones o códigos que le dicen a la computadora qué hacer.<br>
    ■Se puede actualizar, modificar o mejorar con el tiempo.<br>
    ■Es necesario para que el hardware funcione correctamente.<br>
    ■Existen diferentes tipos de software según su función, como sistemas operativos, aplicaciones y programas específicos.<br>
    <img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTyHLtsePPMkMPN1Nz2bdorpLDc8YvHKWXrMQ3Kigne2g&s=10">
<h3 style="color:black;"><p>Uso del Software</p></h3>
<p>-Instalarlo correctamente siguiendo las instrucciones.<br>
    -Leer o conocer su función antes de usarlo.<br>
    -Utilizarlo solo para lo que fue creado.<br>
    -Mantenerlo actualizado para evitar errores o fallas.<br>
    -No descargar software de sitios no confiables.<br>
    -Usarlo con responsabilidad, evitando dañarlo o usarlo de forma incorrecta.<br>
    -Guardar la información correctamente para no perder datos.<br>
